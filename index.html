Trunk

Trunk is a set of graphic components whith some conventions. You can build administration user interfaces extremely fast.

Introduction

Trunk components are independent from third party libraries for several reasons:

  • Support for old browsers is no needed.
  • Should be easy to understand, easy to use and easy to extend.
  • Are really simple.
  • The components rely on the lastests HTML5 standard implementations.
